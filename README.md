# 🚀 AI SQL Documentation Generator

A full-stack web application that converts raw SQL schemas into clean documentation.

## ✨ Features
- Convert SQL schema to documentation
- Column explanations
- Table descriptions
- Responsive UI

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- AI Integration (planned)

## 📸 Demo
Paste schema → Click Generate → Get documentation instantly

## ⚙️ How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev

## 🔐 Environment Variables
Create `.env` file in backend:

OPENAI_API_KEY=your_key_here

## 🚧 Status
Prototype version (rule-based logic, AI integration coming soon)
